INTRODUÇÃO

O Formset é uma plataforma projetada para otimizar e simplificar as pesquisas em campo (Field Survey).

Através de uma interface intuitiva, gestores e usuários têm total autonomia para criar e personalizar suas próprias pesquisas, definindo os parâmetros necessários e mantendo controle absoluto sobre as operações. O sistema permite o monitoramento eficiente de equipes em tempo real, utilizando a captura de GPS para acompanhar as atividades à medida que os dados são enviados.

O objetivo é fornecer uma solução completa para gestão de equipes e operações em campo, oferecendo ferramentas como a criação de questionários dinâmicos, acompanhamento operacional em tempo real e geração de dashboards interativos para apoiar a tomada de decisões estratégicas.
Integrado com ChatGPT, a plataforma permite a criação de questionários com recursos da inteligência artificial, agregando agilidade e inteligência ao processo de coleta de dados. Entre os dados coletados, estão textos, fotos e outras evidências, todos visualizados de forma clara e organizada.
O Formset é a solução ideal para empresas que realizam pesquisas de campo, auditorias, inspeções ou que precisam de controle em tempo real sobre suas operações.

Arquitetura do Sistema

O sistema é composto por três partes principais:
  • Backend: Responsável por processar e gerenciar os dados e operações centrais, utilizando uma API REST para integrar os módulos.
  • Frontend: Oferece uma interface web amigável, moderna e funcional para que usuários e gestores interajam com a plataforma.
  • Mobile: Um aplicativo dedicado, projetado para que equipes em campo realizem as operações diretamente no local, com suporte a envio de dados, images e GPS.
